# Canadian Journal of Linguistics Zotero Style Guide
A CSL reference formatting file for using CJL citation style with Zotero.

My sister is doing her master's degree in Linguistics and got fed up with manually writing out all her citations in this journal's style. I used Claude to generate a .csl file for her to use. I'm uploading this here so any future Canadian linguistics students can find it in the future.

---

## Installation
  1. Download `canadian-journal-of-linguistics.csl` from this repository
  2. In the top left of Zotero, navigate:
     `Edit -> Settings -> Cite -> Add from File...`
  3. Open `canadian-journal-of-linguistics.csl` from your downloads.
  4. `Canadian Journal of Linguistics` should now be an option when choosing your citation style in your document.
